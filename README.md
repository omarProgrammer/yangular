# YAngular - Video Application

## Branches

Each branch represents the module with the AngularJS Foundation application exercises.

## Installation

1. `npm install -g grunt-cli`
2. `npm install -g grunt`
3. `npm install -g bower`
3. `npm install`
4. `bower install`
5. `grunt install`
